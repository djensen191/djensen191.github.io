A Basic Shell Written In C

To explore the workings of a shell similar to Bash I wrote a basic shell, including very simple built in commands. This is an ongoing project. In the future I want to expand this project to continue my exploration and to see what useful functionality can be added. It is written in the C programming language and can be compiled with gcc. A PDF Combination Tool Written In Python3

The command line program PDF toolkit or "pdftk" is a lightweight and free pdf tool that is helpful to allow PDF merging, splitting, and other operations. In this project my goal is to build a simple graphical user interface frontend for the pdftk command line program which takes user provided pdf documents and merges them. This project uses the Python Tkinter framework to develop the GUI and output the commands to the command line using the Subprocess module. Click on the image above to view the GitHub page for the project.

A Convolution Neural Network To Analyze Radio Frequency Fingerprints Written In Python3 and Pytorch

The goal of this project is to take a large open source datatset, the LoRa dataset, and analyze it using a Convolution Neural Network built in Python3 and Pytorch. Though CNN's are typically used for computer vision or image analysis This use of analyzing Radio Frequency signal to find unique characteristics shows promise for research in the area of Internet of Things devices.
